# Steam clone

## Установка

1. Склонируй репозиторий проекта:

`git clone https://github.com/Headsman-4899/Steam_API.git`

2. Перейди в каталог проекта:

`cd Steam`

3. Установи зависимости, используя команду:

`pip install -r requirements.txt`

## Использование

1. Запусти тесты:

`python3 -m pytest`
